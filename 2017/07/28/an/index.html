一.连接外网

VMware提供三种网络连接方式，分别为–桥接、NAT、Host-only三种。

其中，桥接是指直接连接到Host主机的外网中，虚拟机相当于一个独立的主机连接到外网。

NAT和Host-only都是与主机共享一个网络。

我使用的是最简单的桥接方式。但好像NAT方式要更安全。

1.首先打开虚拟机，将网络连接设置为桥接自动模式。
2.关闭虚拟机。
3.鼠标左键点击左上角编辑选项，选择虚拟机网络编辑器。
4.点击更改设置，获得管理员权限。此时会出现桥接模式的Vmnet0，下面的选项也选择桥接，自动。点击确定。
5.打开虚拟机。同时打开主机cmd输入ipconfig命令，得到ip地址。在虚拟机的命令模式中输入ping 主机ip,成功。(虚拟机好像会永久ping下去，按ctrl C停止ping)。ping www.baidu.com也是可以。
