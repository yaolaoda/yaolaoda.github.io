搭建博客心得

发表于 2017-07-08

我这样一个一年来计算机学无所成的小白，对于Git GitHub和Hexo是闻所未闻，即便是艰难的搭建了一个简陋的博客，也是知之甚少。在此记录下遇到的许多障碍，希望之后能给自己的博客添加很多新的功能。

  • 实际上很少使用命令行这种东西，看起来高端大气，但若是不懂指令，一切都是白费力气。在网上查询了比较基本的指令语句，感觉整台电脑的所有操作都可以用命令行完成。
  • 搞不懂git github和hexo之间的关系，在搭建博客中究竟承担着怎样的作用。之后才发现github上注册之后的主题根本不用选，根本还是在hexo，只需要把hexo的博客部署到github上就可以。另外，github的注册要注意库的名字的起名规则。
  • hexo generate,hexo deploy,hexo server,hexo clean还有hexo d -g这些是搭建hexo博客时我经常使用的指令。一开始使用hexo deploy时，出现了“error deployer not found github”的错误，在网上查询后重新发指令npm install hexo-deployer-git –save,无果。最后发现要先修改_config.yml中的type: git。

总之，搭建出来的博客比我想象的还要丑，且功能全无，有很多很多东西需要慢慢去填补，希望一点一点的努力能使我的博客丰满、殷实而有深度。
